settings.json - Innehåller alla lampor som objekt och kan få in en ny lampa när som helst.
                För att lägga till en ny lampa skapa ett objekt i arrayn.
                Exempel:
                        [
                            {
                                "type": "blink",
                                "id": "Lägg till id på lampan"
                            },

                            {
                                "type": "hue",
                                "id": "Lägg till id på lampan",
                                "room": "Lägg till id på rum"
                            }
                        ]
